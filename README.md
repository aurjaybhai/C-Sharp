#### Escape Sequences

| Escape Sequence | Represents            |
|-----------------|-----------------------|
| `\a`            | Bell (alert)          |
| `\b`            | Backspace             |
| `\f`            | Form feed             |
| `\n`            | New line              |
| `\r`            | Carriage return       |
| `\t`            | Horizontal tab        |
| `\v`            | Vertical tab          |
| `\'`            | Single quotation mark |
| `\"`            | Double quotation mark |
| `\\`            | Backslash             |
| `\?`            | Literal question mark |
